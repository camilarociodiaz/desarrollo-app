<h2> Run Locally </h2> 

Clone the project
```
git clone https://github.com/camilarociodiaz/e-book-store.git
```
Go to the project directory
```
cd e-book-store
```
Install dependencies
```
 npm install
 ```
Start the server
```
 npm start
 ```
